# Cinestar Proyecto C#

Este repositorio contiene una arquitectura distribuida para la aplicación web Cinestar:

- **CinestarWebAPI**: API Web construida en .NET 8 que expone los datos de películas y cines.
- **webCinestarConsumirAPI**: Proyecto MVC en .NET 8 que consume directamente la API usando `HttpClient`.
- **webCinestarMVC**: Proyecto monolítico MVC original con la lógica de base de datos directa.

## Cómo empezar
Abre el archivo `Cinestar.slnx` con Visual Studio para cargar los 3 proyectos juntos y compilarlos sin problemas.
